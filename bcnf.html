Implementation Note:
This app is fully implemented in HTML/CSS/Javascript.
Main algorithm is implemented in Javascript.
Please enable your Javascript on your web-browser.
Since it's running on Javascript, browser may freeze on large number of inputs.
It has been tested on Firefox3.6 and Chrome 10.

Instruction:

Note that trivial functional dependencies will not be generated (since they are obvious).
